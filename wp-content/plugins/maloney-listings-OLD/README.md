# Maloney Affordable Listings Plugin

**Version:** 1.0.0  
**Developer:** Responsab LLC  
**Website:** https://www.responsab.com

## Description

Comprehensive listing management system for condos and rentals, built specifically for the Maloney Affordable website.

## Features

- Custom post type for unified listings (condos and rentals)
- Advanced filtering and search functionality
- Interactive map with clustering
- Current rental availability management
- Geocoding and location services
- Integration with Toolset Types
- Custom blocks and shortcodes
- Migration tools for existing data
